
# Ray

A simple to use email sorter named after [Ray Tomlinson](https://en.wikipedia.org/wiki/Ray_Tomlinson).

---


## Setup

Download the repo via a python package manager
```bash
pip install git+https://github.com/Magic-Goose-Software-Co/Ray
```

Run setup command.
```bash
ray
```
and answer its five questions.

---

## Usage

Move some emails into the mailboxes you want it to sort mail into and run `ray`.

---

## Recreate Config

If you need to recreate the configuration for any reason, run:
```bash
ray config
```
